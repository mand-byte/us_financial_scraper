```python
defreward(state):
    r =0

if 持仓中:
# 时间成本（每根K线）
        r -= time_cost

# 收线高于上根K线
if close_now > close_prev:
            r += R1

# 突破持仓新高
if close_now > high_max:
            r += R2

# 触及止损未平仓
if low_now <= stop_loss and agent未平仓:
            r -= P1  # 你的"底线"惩罚

if 主动平仓:
# 按实际盈亏给最终结算奖励
        r += 最终盈亏 / 初始风险R  # 用盈亏比作为结算分

return r
```

---

**这样Agent会学到：**

| 行为           | 结果                       |
| -------------- | -------------------------- |
| 开仓后持续走强 | 持续得分，愿意持有         |
| 开仓后横盘不动 | 时间成本慢慢亏分，考虑离场 |
| 走到止损不平仓 | 重罚，学会底线             |
| 主动在高位平仓 | 结算拿大分                 |

---

**关键参数建议比例：**

```
time_cost : R1 : R2 : P1 : 结算奖励
0.01:0.1:0.2:2.0: 按实际盈亏比
```

P1要远大于其他，才能让止损底线真正有威慑力。
